# ai-foundations

AI and machine learning study notes, experiments, and projects.


## Projects Index

* **[CS311 Oil Price Predictor](./cs311-oil-price-predictor)** - A Python machine learning application using Random Forest Regression to predict the next trading day closing price of Brent crude oil futures.

* **[CS311 Garmin R10 HelpBot](./cs311-ai_helpbot_garmin)** - A Python FAQ chatbot using LangChain, OpenAI embeddings, FAISS vector search, and an LLM API to answer Garmin Approach R10 product-support questions.
  
* **[CS311 CartPole Q-Learning Agent](cs311-cartpole-q-learning)** - A Python reinforcement learning project using tabular Q-learning and Gymnasium CartPole-v1 to train an agent to balance a pole through trial and error.
## About This Repository
This repository contains coursework, lab assignments, and experimental machine learning models built while studying core AI concepts.
